# T00 交易授权与账户控制页面

## 页面定位

当前页面定位为 **交易账户、投资授权和 Agent 自主边界的统一控制页**。

用户在这里决定连接哪个账户、允许交易哪些资产、风险与组合上限、Agent 可以执行到什么程度，以及何时暂停或撤销授权。页面不生成交易建议，不承担行情研究。

## 与 AI 辅助交易台的关系

- AI 辅助交易台只能使用本页已生效的账户和授权。
- 授权变更会使受影响的交易计划、订单草稿和未提交确认失效。
- 已提交订单按用户选择的撤销策略处理，页面不能假装已成交订单可以撤回。
- 页面输出的 `InvestmentMandate` 是所有交易意图的唯一授权依据。

## 页面原型

```text
┌──────────────────────────────────────────────────────────────────────┐
│ 交易授权与账户       当前状态：仿真可用 / 实盘未启用   [暂停 Agent] │
├───────────────────┬──────────────────────────────┬───────────────────┤
│ 3 列：账户与连接  │ 6 列：授权编辑               │ 3 列：只读影响    │
│ ● 仿真账户 正常  │ 当前生效 v3 / 编辑草稿 v4    │ 影响预览 v4       │
│ ○ 实盘 未连接    │ 风险/资产/自主级别/有效期    │ 待重算/已更新     │
│ 权限 / 同步状态  │                              │ 失效计划与草稿    │
│ 章节导航          │                              │ 版本历史          │
├───────────────────┴──────────────────────────────┴───────────────────┤
│ 授权历史与受影响对象        [保存为新版本] [撤销授权]               │
└──────────────────────────────────────────────────────────────────────┘
```

布局约束：

- 1440 px 使用 3/6/3，约 316/648/316 px；
- 右侧影响列只读，必须显示基于哪个草稿版本和计算时点；
- 编辑字段变化后，影响列立即进入“待重算”，不能继续显示旧结果为当前结果；
- 切换账户时，授权编辑和影响列共同清空后重新加载；
- 页面唯一主操作是“保存为新版本”；撤销授权、暂停和风险放宽使用独立确认层。

## 页面功能

| 页面区域 | 功能模块 | 页面功能 | 主要呈现信息 | 用户操作 | AI处理 |
| --- | --- | --- | --- | --- | --- |
| 顶部栏 | 全局状态 | 展示交易能力是否可用 | 仿真/实盘状态、授权有效性、账户异常、Agent 状态 | 查看说明、暂停 Agent | 读取状态，不修改 |
| 顶部栏 | 紧急暂停 | 阻止创建新的交易意图 | 暂停范围、生效时间、未完成订单处理方式 | 暂停、恢复 | 停止生成新计划，不操作既有订单 |
| 账户区 | 账户列表 | 管理仿真和实盘账户 | 账户名称、类型、币种、状态、最后同步时间 | 选择、连接、断开 | 根据选中账户调整解释上下文 |
| 账户区 | 连接授权 | 建立受支持的账户连接 | 经纪商、授权范围、有效期、回调状态 | 发起授权、重新授权 | 不接触账户凭据 |
| 账户区 | 交易权限 | 显示账户可交易范围 | 市场、资产类型、订单类型、时段、只读/可交易 | 查看、申请变更 | 说明权限对计划的影响 |
| 账户区 | 同步状态 | 查看资金与持仓同步 | 资金时点、持仓时点、异常字段、差异 | 刷新、查看差异 | 标记数据是否足以分析 |
| 授权区 | 授权摘要 | 展示当前有效授权 | 目标、期限、风险预算、现金边界、版本 | 查看详情 | 使用授权生成计划提案 |
| 授权区 | 资产范围 | 设置允许和禁止的资产 | 市场、资产类别、白名单、黑名单 | 收紧或编辑范围 | 可解释影响，不能自动扩大 |
| 授权区 | 组合限制 | 设置组合硬边界 | 集中度、行业/地区上限、现金下限、回撤规则 | 编辑、恢复上版 | 评估修改影响，不能确认生效 |
| 授权区 | 单笔限制 | 设置订单级约束 | 单笔金额、单日金额、数量、价格偏离 | 编辑 | 解释限制，不绕过规则 |
| 授权区 | 有效期 | 管理授权有效时间 | 生效、到期、复核日、失效条件 | 延长、缩短、撤销 | 到期前生成提醒 |
| 自主区 | 自主级别 | 选择 Agent 行为范围 | L0–L3 定义、当前可用级别 | 选择、查看差异 | 只能在已选级别内工作 |
| 自主区 | 确认方式 | 设置逐笔或批次确认 | 适用账户、计划范围、确认有效期 | 选择 | 根据方式准备待办，不代确认 |
| 安全区 | 身份验证 | 设置高风险操作验证 | MFA、生物识别、可信设备 | 启用、移除设备 | 不接收验证因子 |
| 安全区 | 活动提醒 | 管理账户安全提醒 | 登录、授权变化、订单、资金与账户变化 | 开启、关闭、选渠道 | 生成提醒摘要 |
| 历史区 | 授权版本 | 查看授权变化 | 版本、变更字段、用户、时间、原因 | 比较、恢复为新版本 | 总结差异 |
| 历史区 | 影响分析 | 查看变更影响 | 失效计划、草稿、待确认订单、策略 | 查看、逐项处理 | 解释原因并给出处理顺序 |
| 全局 | 保存校验 | 创建新授权版本 | 完整性、冲突、风险变化 | 保存、取消 | 可解释冲突；规则服务决定能否保存 |
| 全局 | 撤销授权 | 终止未来交易权限 | 生效范围、未完成订单、恢复条件 | 二次确认撤销 | 停止生成新交易意图 |

## 关键状态

| 状态 | 页面表现 | 用户可执行操作 |
| --- | --- | --- |
| 无账户 | 展示仿真账户创建和实盘连接说明 | 创建仿真账户、连接账户 |
| 账户只读 | 明确持仓可见但不能提交订单 | 查看组合、重新授权 |
| 账户断连 | 显示最后成功同步时点和受影响功能 | 重新连接、继续查看历史 |
| 授权草稿 | 不允许交易台创建可提交草稿 | 补充、保存、放弃 |
| 授权有效 | 展示版本和有效期 | 收紧、暂停、创建新版本 |
| 授权即将到期 | 在顶部和 T01 生成待办 | 复核、延长、撤销 |
| 授权失效 | 阻止新订单，保留历史 | 创建新授权 |
| Agent 已暂停 | 所有页面显示持续状态条 | 恢复、查看受影响任务 |
| 变更冲突 | 并列展示冲突规则 | 修正字段，不静默使用旧值 |

## 开源项目补充

参考 Hummingbot Dashboard 和 FreqUI，首版增加连接健康、最后成功同步、权限摘要、纸面/实盘凭据隔离和只读连接测试。页面不得展示完整 API Key、Secret 或 MFA 因子，也不采用任何默认账号密码。详细依据见 [开源项目参考与页面补充](02_开源项目参考与页面补充.md#51-t00-交易授权与账户控制)。

## AI 与授权边界

| AI可以做 | AI不能做 |
| --- | --- |
| 解释授权字段和风险影响 | 代表用户确认授权 |
| 根据画像生成授权草案 | 自动提高自主级别 |
| 识别字段冲突和遗漏 | 放宽资产范围或风险上限 |
| 总结版本差异 | 保存、延长或撤销授权 |
| 提醒授权即将到期 | 获取账户密码、MFA 或生物信息 |
| 说明暂停对计划的影响 | 绕过已暂停状态继续生成订单 |

## 导航关系

- 入口：用户菜单、T01 授权待办、T03 账户/授权状态、T04 阻断原因。
- 出口：T01 总览、T08 组合、受影响的 T04 计划。
- 返回时保留用户原任务，但必须重新加载授权和账户状态。

## 验收标准

- 默认自主级别为 L0，未确认授权不能创建可提交订单。
- 收紧、暂停或撤销授权立即阻止新的订单意图。
- 授权变化创建新版本，不覆盖历史。
- 受影响的计划和草稿能列出具体失效字段。
- AI 无法直接调用授权生效、撤销或账户连接命令。
- 页面不存储或展示经纪商密码、MFA 密钥和完整敏感凭据。
