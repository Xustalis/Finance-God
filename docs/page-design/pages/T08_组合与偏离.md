# T08 组合与偏离

> 文档类型：页面规格 / Reference  
> 对应导航：组合  
> 适用宽度：1440 px 主画布，1024 px 最低宽度

## 页面任务与主对象

当前页面定位为 **当前持仓、目标组合、风险预算和调仓需求的统一解释页**。

页面以目标和组合为中心，不以单个资产涨跌为中心。用户在这里判断是否需要调整组合，并从偏离创建交易计划。

## 跨页关系

- 用户从持仓或偏离项进入 AI 辅助交易台研究单个标的。
- 多标的偏离生成 T04 交易计划，不直接在本页提交订单。
- T03 和 T07 的实际成交更新组合事实，本页不自行推断持仓变化。

## 信息层级与工作区

```text
┌──────────────────────────────────────────────────────────────────────┐
│ 我的组合 【仿真】 总资产 ¥xxx,xxx  组合数据：10:31  [导入/同步]     │
├──────────────────────────────────────────────────────────────────────┤
│ 12 列摘要：目标进度｜现金｜风险预算｜最大偏离｜数据完整度           │
├───────────────────────────────────────┬──────────────────────────────┤
│ 7 列：当前/目标/计划后配置            │ 5 列：风险/现金/集中度       │
│ 资产 / 地区 / 行业 / 币种             │ 压力摘要 / 硬边界            │
├───────────────────────────────────────┼──────────────────────────────┤
│ 8 列：完整持仓主表                    │ 4 列：单一持久检查器         │
│ 权重 成本 收益 目标权重 偏离          │ 当前对象 / 偏离解释          │
│                                       │ 数据问题 / 时点 / 主操作     │
└──────────────────────────────────────────────────────────────────────┘
```

布局约束：

- 顶部摘要建议高 96 px；中部使用 7/5，底部使用 8/4；
- 右下只有一个检查器，持仓、配置区段和偏离 Chip 共用同一选中对象；
- 检查器头部永久显示最大偏离、硬边界、数据完整度、数据时点和当前对象；
- 数据问题在同一对象检查器内呈现；只有非阻断的深层明细进入抽屉；
- 页面只保留一个纵向滚动，不允许持仓、偏离和数据质量各自滚动；
- “生成交易计划”仅在选中可执行偏离且数据、账户和授权均通过时出现。

## 功能与数据

| 页面区域 | 功能模块 | 页面功能 | 主要呈现信息 | 用户操作 | AI处理 |
| --- | --- | --- | --- | --- | --- |
| 顶部栏 | 账户与模式 | 选择组合账户 | 仿真/实盘、单账户/合并视图、币种 | 切换 | 调整分析上下文 |
| 顶部栏 | 组合时点 | 展示持仓数据新鲜度 | 最后同步、估值时间、来源、延迟 | 刷新、查看来源 | 判断分析是否可用 |
| 顶部栏 | 导入与同步 | 更新持仓数据 | 手动、CSV、账户连接、差异 | 导入、同步、处理冲突 | 识别资产，不能覆盖冲突 |
| 摘要区 | 资产摘要 | 展示组合总体情况 | 总资产、现金、持仓数、当期收益 | 切换周期 | 总结变化 |
| 摘要区 | 目标状态 | 展示目标资金与时间 | 目标金额、日期、进度、预计区间 | 查看目标 | 解释差距 |
| 配置区 | 当前配置 | 展示实际资产配置 | 资产、地区、行业、币种、因子 | 切换维度 | 解释主要暴露 |
| 配置区 | 目标配置 | 展示授权下的目标权重 | 目标权重、允许区间、来源策略 | 查看依据 | 解释策略，不修改目标 |
| 配置区 | 偏离可视化 | 对比当前和目标 | 权重差、金额差、允许范围 | 排序、筛选 | 总结最大偏离 |
| 风险区 | 风险预算 | 展示已使用和剩余额度 | 波动、回撤、集中度、现金下限 | 查看计算 | 通俗解释 |
| 风险区 | 压力情景 | 展示不利市场影响 | 情景、组合损失区间、主要贡献项 | 切换情景 | 生成情景说明 |
| 风险区 | 集中度与重叠 | 识别隐含集中 | 单标的、行业、地区、基金持仓重叠 | 展开穿透 | 解释风险来源 |
| 偏离区 | 偏离列表 | 查看超配、低配和规则偏离 | 维度、当前、目标、阈值、影响 | 选择、忽略提示 | 生成原因摘要 |
| 偏离区 | 不行动影响 | 展示保持现状的结果 | 风险、目标、现金、预计复核日 | 查看 | 生成情景，不制造紧迫感 |
| 偏离区 | 调仓入口 | 从偏离创建计划 | 选中偏离、涉及资产、预计动作 | 生成 T04 计划 | 创建计划草案，不下单 |
| 持仓区 | 持仓列表 | 查看每个持仓 | 数量、可卖、成本、价格、市值、收益、权重、目标 | 排序、筛选、选择 | 解释组合角色 |
| 持仓区 | 持仓详情 | 查看标的在组合中的作用 | 暴露、重叠、风险贡献、研究状态 | 进入 T03 | 传递组合上下文 |
| 现金区 | 现金与待结算 | 区分可用与不可用现金 | 可用、冻结、待结算、计划占用 | 查看明细 | 说明限制 |
| 数据区 | 未解析资产 | 处理无法识别的持仓 | 原始名称、来源、金额、影响 | 匹配、排除、反馈 | 提出候选匹配，不自动确认 |
| 数据区 | 同步差异 | 处理多来源冲突 | 来源值、差异、时间 | 选择权威来源、复核 | 总结冲突 |
| 全局 | 导出 | 导出组合快照 | 版本、时点、币种、来源 | 导出 | 生成摘要说明 |

## 偏离类型

| 类型 | 示例 | 默认处理 |
| --- | --- | --- |
| 权重偏离 | 股票资产高于目标上限 | 生成调仓候选 |
| 现金偏离 | 可用现金低于授权下限 | 阻止新增买入计划 |
| 集中度偏离 | 单一基金或行业超限 | 硬阻断或收紧计划 |
| 目标偏离 | 目标资金进度落后 | 优先调整投入和期限，不直接提高风险 |
| 数据偏离 | 账户与导入持仓不一致 | 暂停受影响分析 |
| 策略偏离 | 实际组合偏离有效策略 | 生成复核任务 |

## 关键状态

| 状态 | 页面表现 | 用户可执行操作 |
| --- | --- | --- |
| 空组合 | 提供手动、CSV 和账户连接入口 | 导入、从零开始 |
| 数据同步中 | 展示来源级进度 | 离开、取消支持的任务 |
| 数据过期 | 冻结旧结论并标记过期 | 刷新、查看历史 |
| 存在未解析资产 | 标记受影响的配置和风险 | 处理匹配 |
| 无需调仓 | 明确显示在允许范围内 | 查看下次复核时间 |
| 需要调仓 | 展示原因和影响，不直接下单 | 生成 T04 |
| 硬约束违规 | 顶部持续警告 | 查看规则、暂停新交易 |
| 账户差异 | 并列来源，暂停合并结论 | 选择权威来源、人工复核 |

## AI 边界

| AI可以做 | AI不能做 |
| --- | --- |
| 解释配置、风险和偏离 | 修改持仓事实 |
| 提出目标组合和调仓计划草案 | 绕过授权改变目标权重 |
| 识别基金持仓重叠和组合角色 | 用短期心智状态提高风险 |
| 总结不行动影响和压力情景 | 把预测当作确定收益 |
| 提议未解析资产匹配 | 未经用户确认覆盖原始数据 |

## 导航关系

- 入口：T01 组合摘要、T07 成交后影响、T09 复盘结果。
- 出口：持仓进入 T03；候选进入 T02；调仓进入 T04；授权问题进入 T00；证据进入 T10。
- 返回时保留账户、配置维度和筛选。

## 验收标准

- 当前、目标和计划后组合使用不同数据系列并标注时点。
- 未解析资产和同步冲突会明确降低结论完整度。
- 目标进度落后不能自动转化为提高风险建议。
- “无需调仓”是有效结果，不强制生成交易机会。
- 实际成交只能根据 `Fill` 更新持仓。
- 任何调仓入口都先创建 T04 计划，不直接生成已提交订单。
