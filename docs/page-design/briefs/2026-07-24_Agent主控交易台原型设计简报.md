# Agent 主控交易台隔离原型设计简报

## 用户与操作环境

- 目标用户：具备投资基础的个人用户与专业交易者。
- 使用场景：进入交易台后同时查看当前交易对象与 Agent 建议；用语言或快捷指令切换左侧
  工作区、筛选数据、填充未提交表单，并查看后台工作流进度和站内提醒。
- 主要设计宽度：1440 px
- 最低支持宽度：1024 px

## 页面任务与主对象

- 页面任务：围绕一个当前标的完成行情查看、持仓核对、策略研究和仿真交易草稿准备。
- 主要对象：当前标的及其版本化页面上下文。
- 对象版本或状态：原型用 `workspace-context/v1` 演示上下文版本；真实行情只接受现有
  PandaData 后端返回，仿真业务区明确标记“仿真”。

## 信息层级与工作区

- 全局产品壳：紧凑顶部报头，显示 `FINANCE GOD`、仿真模式、PandaData 状态、请求间隔、
  提醒入口和用户设置。
- 页面工具栏：左侧工作区使用文字导航承载信息、持仓、自选、交易记录和钱包；信息工作区
  再提供行情、交易、策略互斥状态。
- 主工作区：当前标的、行情/图表状态和当前工作态是首要视觉层；左侧不是全局图标导航。
- Inspector：不另建重复 Inspector；当前对象摘要并入主工作区标题和表格。
- 常驻 AI 侧栏：右侧默认展开，显示当前上下文、对话、工作流状态、输入框和恰好三条动态
  快捷指令；完成后的工作流步骤自动折叠且可展开。

## 视觉与交互命题

- 视觉命题：一张暖米色财经编辑台，深墨规则线将高密度交易事实与一条常驻编辑注释栏并置，
  不使用卡片墙、蓝色强调或悬浮聊天气泡。
- 内容计划：产品与状态报头 → 左侧工作区导航 → 当前对象主图/主表 → 右侧 Agent 对话、
  工作流和快捷指令 → 站内提醒。
- 交互命题：
  - 工作区切换以 120–180 ms 的墨色下划线和内容淡入表达，不进行全页翻版。
  - Agent 执行安全 UI 动作时，左侧目标区域短暂显示 180 ms 的规则线反馈。
  - 工作流运行时逐步更新；完成后折叠为摘要，用户可用原生 `details` 展开。

## 数据与失败状态

| 数据 | 来源 | 实际频率 | 新鲜度判断 | 失败状态 |
| --- | --- | --- | --- | --- |
| 当前行情 | 后端 `/api/market/overview` → PandaData | 由响应 `frequency` 给出 | 使用 `provider_time` 与 `freshness` | 不显示演示价格；保留空值并显示错误与重试 |
| 页面请求 | 原型统一请求控制器 | 默认 60 秒，可手动刷新 | 单独显示最近请求时间 | 页面隐藏时停止，恢复后立即请求 |
| 持仓、订单、钱包 | 原型结构数据 | 仅用于布局演示 | 明确标记“仿真原型” | 不声称来自真实账户或成交 |
| Agent 工作流 | 原型交互状态机 | 用户指令触发 | 显示每一步状态和运行时间 | 显式失败；不生成交易事实 |
| 提醒记录 | 原型内存记录 | 事件触发 | 显示触发时间与已读状态 | 真实版本由后端通知表和事件消费器接管 |

市场数据必须来自 PandaData。账户、持仓、计划、订单、成交和复盘必须标记为仿真数据。

## 主操作与启用条件

- 唯一主操作：当前工作态为“运行当前分析”或“进入最终复核”；原型只演示前者。
- 启用条件：存在当前对象；真实实现还需要行情版本、账户、授权和能力状态有效。
- 禁用原因：行情不可用/过期、无仿真账户、Agent 暂停、工作流运行中或后端能力阻断。
- 成功后改变的对象：只新增版本化 Agent 产物或更新未提交 UI 上下文；不修改订单、持仓、
  钱包或账本事实。

## Agent 可控动作边界

- 可控：切换信息/持仓/自选/交易记录/钱包，切换行情/交易/策略，选择标的，调整展示周期，
  填充未提交订单草稿，聚焦指定表单字段。
- 不可控：打开或修改用户设置、保存凭据、进入或确认 T06、提交/撤销订单、修改持仓/现金、
  标记必要风险提醒为已处理。
- 所有可控元素必须暴露稳定 `data-agent-action`、`data-agent-object` 和可读描述；右侧只调用
  注册表中的白名单动作，不进行任意 DOM 点击。

## 影响范围

- 受影响路由：无生产路由；隔离原型对应未来 T03 `/desk`。
- 受影响组件、composable 或 store：仅 `frontend/prototypes/agent-workbench/` 内新增文件。
- 1440 px 行为：左侧工作区占可用宽度，右侧 Agent 约 380 px 默认展开。
- 1024–1279 px 行为：右侧 Agent 首次访问默认收起为 44 px 可见窄栏，用户可主动展开；
  左侧导航缩短说明文字，关键行情时点和仿真标签不隐藏。
- 小于 1024 px 的桌面访问提示：整页显示明确桌面宽度提示，不创建移动布局。

## 明确不做

- 不修改现有生产前端、后端、数据库迁移或 API。
- 不提供实盘交易、自动交易或 Agent 自动提交/撤单。
- 不硬编码或模拟 PandaData 行情；后端不可用时展示失败。
- 不把每次行情轮询都升级为 AI 工作流；原型仅展示“阈值事件触发工作流”的交互结果。
- 不实现真实自然语言模型、真实后台调度、SSE/WebSocket、通知消费者或多渠道推送。
