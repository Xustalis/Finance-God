## 0. 设计简报（减法后）

- 用户：具备投资基础的个人用户与专业交易者，共用一套桌面工作区；
- 设备：1440 px 主设计宽度，1024 px 最低宽度，不设计移动端；
- 任务：每个页面只完成唯一主要任务（当前对象的事实展示 + 单一主操作）；
- 数据：行情和数据目录来自 PandaData；账户、持仓、计划、订单、成交和复盘为仿真；
- 视觉：浅色、紧凑、数据优先、专业交易工作区（无卡片墙、无营销元素）；
- AI：所有页面常驻同一右侧栏，可收起为 44 px 窄栏并保存选择；
- 生产基线：Vue 3、TypeScript、Vite、Vue Router、Pinia、自有组件层和
  `lightweight-charts`；当前 `prototype/` 为历史设计验收原型。

**减法说明**：前端功能已精简 60-70%，复杂度已全部下沉至后端。页面只保留必需信息和唯一主操作。

## 10. 最终结论

- 验收结论：**通过**；
- 未通过项目：无；
- 已知限制：
  - 账户、持仓、计划、订单、成交和复盘仍为仿真，不连接券商；
  - PandaData 对具体市场和标的的可用性以 Provider 返回为准；例如本次
    `00700.HK` 未返回记录，界面会明确显示错误；
  - 当前 SVG 图表和无构建原型仅用于该历史记录；生产实现按 Vue 3、TypeScript、
    Vite 和 `lightweight-charts` 基线构建；
- 复现命令：

```bash
cd backend
PYTHONPATH="$PWD/vendor/verifolio-unified-agents-0.2.0/agent_framework/src" \
  .venv/bin/python -m unittest discover -s tests -v
.venv/bin/python -m uvicorn server:app --host 127.0.0.1 --port 4173
```

浏览器访问 `http://127.0.0.1:4173/`。
