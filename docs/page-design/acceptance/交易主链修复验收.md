## 10. 最终结论

- 验收结论：核心交易主链通过，整页规范验收仍为失败。身份、事务、真实行情、订单生命周期、三档宽度和共享 AI 侧栏均已闭环。
- 未通过项目：面板拖拽调宽、完整键盘焦点流程、全局 reduced-motion 复验，以及外部字体/favicon 控制台资源错误。
- 已知限制：收盘超过 120 秒后分钟行情按真实语义显示 stale，交易入口保持禁用；这是数据门禁的预期行为。

  - 后端静态检查：`.venv/bin/python -m ruff check app/ finance_god/`
  - 前端：`npm test -- --run && npm run type-check && npm run lint && npm run build`
