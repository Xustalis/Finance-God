## 10. 最终结论

- 验收结论：通过。真实分钟时间、默认批量和未知标的部分失败合同均已闭环。
- 未通过项目：无。
- 已知限制：收盘超过 120 秒后行情按真实语义显示 stale，订单门禁保持禁用；未用摄取时间伪装新鲜。
- 复现命令或测试入口：`uv run pytest tests/market_data -q -m "not live"`、`RUN_PANDADATA_LIVE_SMOKE=1 uv run pytest tests/market_data/test_live_smoke.py -q`、`npm test -- --run src/tests/market-defaults.spec.ts`。
