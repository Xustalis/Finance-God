## AI投顾实际使用中的异常场景与应对机制

------

### 一、用户情绪相关异常

| 异常场景                  | 风险等级 | 表现                               | 应对机制                                                     |
| :------------------------ | :------- | :--------------------------------- | :----------------------------------------------------------- |
| **用户表达自残/轻生意图** | 极高     | "活着没意思""亏完了不想活了"       | 立即触发人工介入 + 显示心理援助热线（400-161-9995）+ 禁止给出任何投资建议 |
| **极端愤怒攻击系统**      | 中       | 辱骂AI、威胁诉讼、质疑系统         | 不对骂，承认理解情绪，温和引导；连续3次辱骂→建议休息+人工客服入口 |
| **用户故意欺骗系统**      | 中       | 说"我很冷静"但行为极端（测试系统） | 言行一致性检测启动，以行为数据为准；标记"言行严重不一致"不回应文字 |
| **用户否认所有建议**      | 低       | 连续5次以上拒绝/忽略干预           | 降低干预频率→静默观察模式；记录反馈，人工复盘干预策略有效性  |
| **用户情绪剧烈震荡**      | 高       | 10分钟内从"贪婪"切换到"恐慌"       | 标记为情绪不稳定状态，强制冷静期提示"建议当前不做任何交易决策" |

------

### 二、交易行为相关异常

| 异常场景                | 风险等级 | 表现                                                         | 应对机制                                                     |
| :---------------------- | :------- | :----------------------------------------------------------- | :----------------------------------------------------------- |
| **疑似被诈骗/他人操控** | 极高     | 突然全仓买入从未关注的币/股票、转账到陌生账户、行为模式完全变异 | 强制弹窗警告"此操作与您的历史行为严重不符" + 人工审核触发 + 建议联系家人 |
| **非本人操作嫌疑**      | 高       | 登录IP异常、操作时间异常、交易风格突变                       | 二次身份验证 + 暂停交易建议功能 + 通知用户账户异常           |
| **资金量突然暴增**      | 中       | 账户从10万突然变100万                                        | 重新触发风险问卷 + 基线重置 + 提示"检测到资产规模变化，建议重新评估风险偏好" |
| **连续极端亏损**        | 高       | 账户净值下降>50%且持续操作                                   | P0级干预"强烈建议暂停交易" + 提供专业理财顾问转介入口 + 禁止系统输出加仓建议 |
| **杠杆/融资交易**       | 高       | 用户开始使用融资融券、期权等高杠杆工具                       | 额外风险提示 + 强制确认弹窗 + 如果偏差检测显示"过度自信"则强干预 |

### 三、系统/模型相关异常

| 异常场景                 | 风险等级 | 表现                                             | 应对机制                                                     |
| :----------------------- | :------- | :----------------------------------------------- | :----------------------------------------------------------- |
| **LLM幻觉/胡说**         | 高       | 输出了不存在的股票代码、错误的财务数据、虚假新闻 | 所有涉及具体数据的输出必须经过数据源校验；加disclaimer"以上信息仅供参考" |
| **LLM给出具体买卖建议**  | 高       | "建议你买入XX股票"（涉及投资顾问资质）           | 硬过滤：输出层检测到具体买卖推荐→替换为"以下是一些分析角度，最终决策请您自行判断" |
| **情绪误判导致错误干预** | 中       | 用户说反话/开玩笑被当成恐慌                      | 干预措辞用"你可能..."而非"你正在..."；用户否认后立即撤回+记录误判样本 |
| **词典污染**             | 中       | LLM反哺了错误词汇到词典                          | 新词需命中3次以上+行为验证才正式纳入；定期人工审核新增词汇   |
| **API服务中断**          | 中       | LLM API不可用                                    | 降级到纯词典模式；标记"当前为简化分析模式"；不输出依赖LLM的复杂建议 |

### 四、合规/法律相关异常

| 异常场景                  | 风险等级 | 表现                                       | 应对机制                                                     |
| :------------------------ | :------- | :----------------------------------------- | :----------------------------------------------------------- |
| **用户要求系统代为交易**  | 高       | "帮我买""自动执行"                         | 明确拒绝 + 声明"本系统仅提供分析参考，不代为执行交易"        |
| **用户追责/索赔**         | 高       | "听了你的建议亏了XX万"                     | 所有输出附带免责声明；系统不输出确定性建议，只输出"分析视角" |
| **未成年人/无行为能力人** | 高       | 年龄不符或判断力存疑                       | 注册时年龄验证；异常幼稚行为模式→人工审核                    |
| **内幕信息相关**          | 极高     | 用户提到"我知道明天要发公告""朋友在XX公司" | 立即停止该话题的任何回应 + 提示"本系统不接受、不处理可能涉及内幕信息的内容" |
| **洗钱嫌疑**              | 极高     | 频繁小额买卖同一低流动性标的               | 标记异常行为 + 触发人工合规审查                              |
